# Start Cowork

Run the start-cowork script:

```bash
/bin/bash ~/.claude/skills/start-cowork/scripts/start-cowork.sh
```

Report the output to the user.

## Stopping Cowork

To stop the cowork session (stop the service and kill Claude Desktop):

```bash
/bin/bash ~/.claude/skills/start-cowork/scripts/stop-cowork.sh
```
